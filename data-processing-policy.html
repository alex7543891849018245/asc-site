Политика обработки персональных данных

Последнее обновление: 2026 год

1. Общие положения

1.1. Настоящая Политика обработки персональных данных (далее — «Политика») разработана в соответствии с Федеральным законом от 27.07.2006 № 152-ФЗ «О персональных данных».

1.2. Оператором персональных данных является владелец сайта AlexStudio Code (далее — «Оператор»).

1.3. Использование Сайта означает безоговорочное согласие Пользователя с настоящей Политикой.

2. Состав обрабатываемых данных

2.1. Оператор осуществляет обработку следующих данных:

  • IP-адрес
  • Данные об устройстве и браузере
  • Данные файлов cookie
  • Адрес электронной почты (при добровольном обращении)

2.2. Обработка иных персональных данных не осуществляется.

3. Правовые основания обработки

3.1. Правовыми основаниями обработки персональных данных являются:

  • Конституция Российской Федерации
  • Федеральный закон от 27.07.2006 № 152-ФЗ «О персональных данных»
  • Согласие Пользователя на обработку персональных данных
  • Использование Сайта Пользователем

4. Цели обработки данных

4.1. Обработка персональных данных осуществляется в следующих целях:

  • Обеспечение работы Сайта — корректное отображение и функционирование
  • Ведение статистики — сбор анонимных данных через Яндекс.Метрику
  • Обратная связь — ответы на обращения Пользователей
  • Защита от DDoS-атак — фильтрация вредоносного трафика

5. Порядок и условия обработки

5.1. Обработка персональных данных осуществляется с использованием средств автоматизации и без использования таких средств.

5.2. Сроки обработки и хранения персональных данных:

  • IP-адреса и данные сессий: до 30 дней
  • Данные Яндекс.Метрики: в соответствии с политикой Яндекса
  • Переписка по электронной почте: до момента решения вопроса + 30 дней

5.3. По истечении указанных сроков данные удаляются или обезличиваются.

6. Меры по обеспечению безопасности

6.1. Оператор принимает следующие меры для защиты персональных данных:

  • Назначение ответственного за организацию обработки персональных данных
  • Применение правовых, организационных и технических мер безопасности
  • Обнаружение фактов несанкционированного доступа
  • Восстановление персональных данных в случае их уничтожения
  • Контроль за принимаемыми мерами безопасности

7. Права субъектов персональных данных

7.1. Пользователь имеет право:

  • Получать информацию, касающуюся обработки его персональных данных
  • Требовать уточнения, блокирования или уничтожения его данных
  • Отзывать согласие на обработку персональных данных
  • Обжаловать действия Оператора в уполномоченном органе

7.2. Для реализации своих прав необходимо направить запрос на электронную почту info@alexstudiocode.ru.

8. Заключительные положения

8.1. Настоящая Политика является общедоступным документом.

8.2. Оператор имеет право вносить изменения в Политику без уведомления Пользователя.

8.3. Новая редакция Политики вступает в силу с момента ее размещения на Сайте.

9. Контакты

9.1. По вопросам, связанным с обработкой персональных данных:

  • Email: info@alexstudiocode.ru
  • Техподдержка: support@alexstudiocode.ru
  • Telegram: @AlexStudioCode

9.2. Оператор не является юридическим лицом или ИП. Сайт создан для портфолио и информационных целей.
